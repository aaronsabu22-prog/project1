# AI Research Paper Assistant

## Overview

AI Research Paper Assistant is a Python-based web application that helps users analyze research papers efficiently. Users can upload PDF research papers, generate concise summaries, ask questions about the content, and perform semantic searches using AI.

## Features

- Upload PDF research papers
- Extract text from PDFs
- AI-generated summaries
- Question Answering
- Semantic Search
- FastAPI backend
- Responsive web interface

## Technologies Used

- Python
- FastAPI
- HTML
- CSS
- JavaScript
- PyMuPDF
- Hugging Face Transformers
- Sentence Transformers
- Scikit-learn
- NumPy
